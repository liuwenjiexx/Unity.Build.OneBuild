# Install

## 安装

**仅支持 Android 设备**

1. 点击菜单 `Build/Install` 打开安装窗口
2. 选择 `Device`
3. 安装包列表会显示 `[Project]/Build` 目录下所有 `.apk` 文件，按文件最后修改时间排序，选择安装文件
4. 点击 `install` 按钮 安装到设备



## Device 下拉列表没有设备排除以下原因

- 点击右边刷新按钮
- 手机是否正常连接电脑
- 检查手机是否开启调试模式
- 是否配置 ANDROID_HOME
- 检查 `adb` 命令，`Win+R` 输入 `cmd`，运行 `adb devices`，是否能显示设备
